本站放置重要结论。



带拉格朗日余项的 $n$ 阶泰勒公式

$$ f_{x} = f_{x_{0}} + {f}^{'}_{x_{0}}(x-x_{0}) + \frac{ {f}^{''}_{x_{0}} {(x-x_{0})}^{2} }{2!} + \cdots + \frac{ {f}^{(n)}_{x_{0}}(x-x_{0})}{n!} + \frac{ {f}^{(n+1)}_{\xi}(x-x_{0})}{(n+1)!}$$

也可写作

$$f_{x} = \displaystyle \sum_{i = 0}^{n} \frac { f_{x_{0}}^{{(i)}} { {(x - x_{0}) }^{i} } }{i \, !} + \frac{ {f}^{(n+1)}_{\xi}(x-x_{0})}{(n+1)!}$$
